# TIC-TAC-TOE FRONTEND

A frontend project created for IA college classes.

Here you'll find the interface of the game, allowing the user to play it.

### Requirements

- Node.js

### Installing

To install all dependencies run:

```bash
npm install
```

### Running

On terminal:

```bash
npm run dev
```

OBS: Once the command is run, the nuxt engine takes a while do load
